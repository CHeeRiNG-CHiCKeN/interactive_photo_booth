# AI Photo Booth — 아티스트 포즈 추천 모드

Python + OpenCV + MediaPipe를 활용한 인터랙티브 AI 포토부스 프로토타입입니다.  
실시간 웹캠 영상에서 사용자의 전신 포즈와 손 관절을 추적하고, 화면 옆에 성인 남성 스틱피겨 아티스트가 나타나 함께 포즈를 취합니다.

---

## 실행 방법

### 의존성 설치

```bash
pip install opencv-python mediapipe numpy Pillow
```

### 실행

```bash
python project1.py
```

### 조작키

| 키 | 동작 |
|----|------|
| `Space` | 포즈 즉시 전환 |
| `Q` | 종료 |

## 주요 기능

### 아티스트 캐릭터
- 성인 남성 7.5등신 비율 (어깨 2.8H · 몸통 3.0H · 팔 2.7H)
- 사용자의 카메라 거리에 맞게 크기 자동 조정
- 사용자보다 눈높이가 높게 배치
- 호흡 애니메이션 (전신 사인파 흔들림)
- 랜덤 눈 깜빡임 + EMA 시선 추적

### 포즈 4종 (10초마다 자동 전환)

| 포즈 | 동작 |
|------|------|
| 반쪽 하트 | 왼팔로 아치를 그려 사용자 어깨 방향으로 뻗기. 손가락으로 반쪽 하트 제스처 |
| 머리 쓰다듬기 | 사용자 머리 위를 1.6초 주기로 왕복하며 쓰다듬기 |
| V 사인 | 오른팔로 가슴 앞에서 브이 (사용자 시야를 가리지 않음) |
| 어깨동무 | 왼팔이 사용자 어깨 위를 넘어 얹히는 베지어 아치 |

### 기술 구현

- **MediaPipe Holistic** — 전신 포즈 + 양손 손가락 관절 동시 감지
- **EMA 팔 스무딩** (`ArmState`) — 포즈 전환 시 이전 위치에서 자연스럽게 이어짐
- **베지어 곡선 팔** — 반쪽 하트·어깨동무에서 자연스러운 아치 형태
- **페이즈 구조** — 중립 복귀 (0.6s) → 인트로 (2.0s) → 포즈 유지
- **한글 렌더링** — PIL + 맑은 고딕으로 깨짐 없이 출력
