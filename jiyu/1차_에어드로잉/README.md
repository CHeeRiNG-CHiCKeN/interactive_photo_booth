# AI Photo Booth — 에어드로잉 & 그림 이동

Python + OpenCV + MediaPipe를 활용한 손가락 공중 드로잉 프로토타입.  
웹캠 앞에서 오른손 제스처로 허공에 그림을 그리고, 집어서 이동가능

---

## 실행 방법

### 의존성 설치

```bash
pip install opencv-python mediapipe numpy
```

### 실행

```bash
python ai_photo_booth.py
```

---

## 조작법

| 제스처 / 키 | 동작 |
|-------------|------|
| 검지만 펴기 | 그리기 모드 — 검지 끝으로 공중에 드로잉 |
| 엄지 + 검지 끝 모으기 | 조작 모드 — 그린 그림을 집어서 이동 |
| `C` | 전체 캔버스 초기화 |
| `Z` | 마지막 스트로크 취소 (Undo) |
| `Q` / `ESC` | 종료 |

## 주요 기능

### 손 인식
- **오른손 전용** 인식 (양손이 화면에 있어도 오른손만 사용)
- MediaPipe 21개 랜드마크 기반 제스처 감지
- 주먹을 핀치로 오인식하지 않도록 MCP 관절 기준 필터 적용

### 드로잉
- **EMA 스무딩**: 손 떨림을 부드럽게 완화 (계수 0.35)
- **최소 이동거리 필터**: 5px 미만 움직임은 무시하여 미세 떨림 제거
- **모드 전환 지연**: 8프레임 동안 비-그리기 자세가 유지되어야 선을 끊음 (오인식 방지)
- 스트로크마다 색상 자동 순환 (하늘색 → 보라 → 연두 → 파랑 → 코랄 → 노랑)

### 시각 효과
- **글로우 효과**: Gaussian Blur 3단계(11px · 25px · 51px)를 합성하여 빛나는 선 표현
- **파티클**: 그리기 중 손끝에서 빛 파티클 발생 (중력 적용, 수명에 따라 투명해짐)

### 오브젝트 조작
- 핀치 제스처로 가장 위에 있는 스트로크를 선택
- 선택된 스트로크는 경계 박스로 표시
- 핀치를 유지하며 손을 움직이면 스트로크 이동
